# Analytics Snapshot 📈

A modern **business analytics widget** — rebuilt as a showcase by **[Governor Studio](https://governor-studio.vercel.app)**.

## Features
- 📊 KPIs (visitors, revenue, conversion) with **count-animated** values and trend deltas
- 📈 Animated sparkline that redraws per period
- 🔁 7d / 30d / 12m period toggle
- 📱 Responsive, accessible, reduced-motion aware

## Stack
**Vite · React 19 · Tailwind CSS v4 · Framer Motion**

## Run locally
```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build → dist/
```

Rebuilt from a static challenge into a real analytics widget to demonstrate Governor Studio's delivery.
